Detailed Description

Macro Documentation

OPENVISA_VERSION

OpenVisa 版本号字符串.

OPENVISA_VERSION_MAJOR

OpenVisa 主要版本号，该版本号变化表示 OpenVisa 的功能或API发生重大变化或达到某设定目标.

OPENVISA_VERSION_MICRO

OpenVisa 末端版本号，该版本号变化表示 OpenVisa 的bug修复，功能和API未发生变化.

OPENVISA_VERSION_MINOR

OpenVisa 次要版本号，该版本号变化表示 OpenVisa 的功能或API发生变化.
